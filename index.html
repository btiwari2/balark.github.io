Digital glass forming with learning-based control

Learning-based control · VLM/VLA · ILC Advanced manufacturing · Real-time control

Real-time thermal control for volumetric laser-heated digital glass forming, combining system identification, z-domain tracking control, iterative learning control, and vision-language-action policies for anomaly-aware adaptation.

Built a real-time C++/rt-preempt Linux controller for volumetric laser-heated digital glass forming, fusing thermal, visual, and motion feedback to achieve precise temperature tracking and robust defect reduction.

Vision-based digital metal forming

Computer vision · Iterative learning control

Vision-based tracking of tool deflection and part morphology in dieless digital metal forming, using computer vision, force, and motion feedback to enable adaptive springback compensation with iterative learning control.

Created an OpenCV-based pipeline for tool deflection and part shape estimation, integrated with an iterative learning controller to iteratively update forming trajectories and compensate for springback in dieless forming.

ISF metal embossing & sheet forming

Manufacturing automation · Motion & sensing

Integrated a Schunk Alpha 2D linear stage with Yaskawa Σ-7 servo motion for coordinated 3-axis embossing and incremental sheet forming (ISF). Closed the loop with quadrature differential encoders on NI DAQmx, added Botasys MegaONE force data over RS232 for force control, and converted CAD/front-surface geometry into executable paths— producing a leaf motif and an embossed pattern based on Notre Dame’s Main Building on sheet metal.

  • Incremental sheet forming (ISF)
  • Yaskawa Sigma-7
  • Differential ABZ Encoder
  • Variable Frequency Drive
  • CAD to path
  • System identification
  • Tracking control

CUDA PIC two-stream instability

GPU computing · Plasma simulation

Developed a CUDA-accelerated (C/C++) electrostatic particle-in-cell (PIC) code simulating the two-stream instability with 40,000 particles. Custom GPU kernels for the Jacobi Poisson solver, grid-to-particle field interpolation, and thread-synchronized iterative convergence achieved a 12x speedup over the serial CPU implementation.

  • C

  • CUDA
  • GPU computing

KUKA iiwa 7-DOF manipulator control

Robotics · Nonlinear control

As part of AME-50551 (Robotics Course at ND), modeled a 7-DOF KUKA iiwa arm using URDF and Denavit-Hartenberg parameters, built forward kinematics via homogeneous coordinate transformations, and implemented custom numerical inverse kinematics in SE(3) using the manipulator Jacobian and damped least-squares. Designed smooth joint-space trajectories from task-space waypoints and used a computed-torque (model-based) controller, leveraging full robot dynamics (mass matrix, Coriolis, gravity) from MATLAB's Robotics System Toolbox, to accurately track paths over an aircraft fuselage and around a bowl rim. The project demonstrates end-to-end capability in manipulator modeling, kinematics, trajectory generation, and nonlinear control, with results visualized as 3D simulations suitable for deployment to real hardware.

  • Robot kinematics
  • Inverse kinematics
  • Trajectory planning
  • Computed-torque control
  • Robot simulation
  • MATLAB
  • URDF
