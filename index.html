Projects

Selected robotics and controls projects spanning digital forming, human-centered systems, and autonomous platforms. Each project below is designed to be paired with short videos or keyframes.

Digital glass forming

Advanced manufacturing · Real-time control

Built a real-time C++/rt-preempt Linux controller for volumetric laser-heated digital glass forming, fusing thermal, visual, and motion feedback to achieve precise temperature tracking and robust defect reduction.

  • Real-time control
  • Thermal imaging
  • Reinforcement learning
  • VLM/VLA

Digital metal forming

Computer vision · Iterative learning control

Created an OpenCV-based pipeline for tool deflection and part shape estimation, integrated with an iterative learning controller to iteratively update forming trajectories and compensate for springback in dieless forming.

  • Computer vision
  • Iterative learning control
  • EtherCAT

User intent recognition for EksoGT

Human-in-the-loop control · Gait analysis

Implemented stacked LSTM classifiers on EksoGT IMU and force sensor data to predict user intent across assistive and control conditions, with analysis of feature importance and performance across subject groups.

  • LSTMs
  • Time-series analysis
  • Exoskeleton control

Research

My research focuses on intelligent manufacturing robotics, combining real-time control, learning, and perception for digital glass and metal forming processes and human-centered systems.

Digital glass forming with learning-based control

Real-time thermal control for volumetric laser-heated digital glass forming, combining system identification, z-domain tracking control, iterative learning control, and vision-language-action policies for anomaly-aware adaptation.

  • Real-time control
  • Reinforcement learning
  • VLM/VLA
  • Iterative learning control

Vision-based digital metal forming

Vision-based tracking of tool deflection and part morphology in dieless digital metal forming, using computer vision, force, and motion feedback to enable adaptive springback compensation with iterative learning control.

  • Computer vision
  • Iterative learning control
  • EtherCAT

User intent recognition for exoskeletons

LSTM-based intent recognition for EksoGT exoskeletons using IMU and force sensor streams, supporting human-in-the-loop control for assistive gait.

  • LSTMs
  • Human-robot interaction
  • Time-series modeling
